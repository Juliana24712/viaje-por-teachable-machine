Viaje por teachable machine

1Funciones de la interfaz gráfica de teachable machine

Las clases: Las clases en Machine learning, exactamente en problemas de clasificación, son los objetos que se predicen según una entrada o conjunto de variables. Las clases en teachable machine, poseen dos entradas por cámara o archivos en la nube. En cámara vemos una configuración para FBS significan los fotogramas por segundo. El retraso, cuantos segundos configurados para empezar a tomar los elementos y duración se refiere al tiempo que el computador se demora tomando fotos. Para potenciar la calidad de las clases, se requiere datos precisos con correcta correlación entre ellos.

2Preparación de modelo

Advanced (Épocas): Número de veces que el algoritmo recorre todo el conjunto de datos. Entre más épocas más aprendizaje de la máquina. Tamaño del lote: Se utiliza para datos en grande cantidad. Tasa de aprendizaje: Rapidez con que el modelo aprende. Entre más se entrene el modelo de mejor manera reconoce los elementos.

3Análisis de elementos en teachable machine

Al aumentar el número de épocas a 60 épocas y la tasa de aprendizaje a 0,00111, el modelo reconoce mejor la imagen y la probabilidad es más acertada y mayor. Por el contrario, al reducir el número de épocas a 40 y la tasa de aprendizaje a 0,000001, el sistema no alcanza a predecir ni el 50% de la probabilidad y se confunde con otros personajes. Del mismo modo, el modelo de clasificación de sonidos funciona de mejor manera, cuando se le introducen más conjuntos o datos de entrada precisos. Igualmente, entre mayor valor de tasa de aprendizaje más alta es la probabilidad acertada por cada clase. Finalmente, en el modelo de clasificación de posturas, se debe tomar muestras en la que la persona aparezca totalmente para que el modelo capte los puntos de los brazos o cara y lo identifique de mejor manera en un usuario diferente. Por el contrario, si las clases no presentas datos de entrada con muestras identificando los puntos, el modelo no reconoce las posiciones de entrada, ya sea inclinación a la derecha o brazo arriba.

4Machine learning

¿Qué es machine learning? Machine learning se refiere al conjunto de técnicas que se usan para que un computador tome decisiones a partir de datos. Según la estructura del pensamiento del ser humano, las decisiones para el aprendizaje de máquina (Remember – formulate - predict framework) consiste en estos tres pasos: 1. We remember previous data 2. We formulate a general rule 3. We use that rule to make predictions about the future. La experiencia de nosotros es equivalente a los datos del computador. Nuestra regla general es equivalente al algoritmo, y la decisión es equivalente a la predicción.
